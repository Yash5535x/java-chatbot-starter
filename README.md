# Java Rule-Based Chatbot (Console)

**Project:** Rule-Based Chatbot

**Description:**
A small, easy-to-understand Java console chatbot that responds to user input using simple keyword rules. Good as a beginner project: demonstrates Java I/O, string handling, basic control flow, and simple project structure.

**Features:**
- Keyword matching responses
- Fallback responses when no keyword matches
- Small conversation history saved to `chat_history.txt`
- Simple `commands`: `exit`, `help`, `history`

**Requirements:**
- Java 11 or newer

**Run:**
javac -d out src/com/chatbot/*.java
java -cp out com.chatbot.Main
